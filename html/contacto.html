Informacion de contacto

Bienvenido al Sistema de Ventas.
Esta aplicación permite consultar productos y proveedore asi como visualizar detalles e informacion de contacto respectivamente.
Tambien se podra visualizar aquellos productos que estan en oferta actualmente, en la seccion correspondiente.
Fue desarrollada utilizando HTML, CSS y JavaScript, proporcionando una interfaz intuitiva y fácil de utilizar.
El objetivo principal es facilitar la consulta y visualización de información de manera eficiente. Este proyecto es desarrollado para evidencia de conocimientos adquiridos en la materia de 'Fundamentos de Tecnología Web'.
